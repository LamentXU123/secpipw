English 中文

secured_pip：避免 PyPI 供应链攻击，最好用也最省事的方式。

$

不用学习，

不用配置，

不用改变使用习惯，

不用承受性能损失，

装上 spip，像原来的 pip 一样用，

但会更加 安全。

开始安装。

先安装 secured_pip，再用 spip 命令，你原来的工作流基本不用变。

PowerShell（Windows）

pip install secured_pip
if (!(Test-Path $PROFILE)) {
  ni $PROFILE -Force
}
'function pip { spip @args }' >> $PROFILE
. $PROFILE

Bash（Linux）

pip install secured_pip
echo "alias pip='spip'" >> ~/.bashrc
source ~/.bashrc

Zsh（macOS）

pip install secured_pip
echo "alias pip='spip'" >> ~/.zshrc
source ~/.zshrc

尽量无痛地帮你避开 LiteLLM、dspy、colorama 等这类供应链投毒风险。
