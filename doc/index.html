Let's install.

Install secured_pip, then use the spip command. Best to alias pip to spip to keep your usual workflow.

Command Prompt (Windows)

pip install secured_pip
doskey pip=spip $*

Bash (Linux)

pip install secured_pip
echo "alias pip='spip'" >> ~/.bashrc
source ~/.bashrc

Zsh (macOS)

pip install secured_pip
echo "alias pip='spip'" >> ~/.zshrc
source ~/.zshrc

That's it.

Now you are much more safer when you type pip install :)
